## 重新扫描

重新扫描插件文件夹下的所有插件。

## 打开脚本文件夹

打开脚本文件夹，即 `C:\Users\***\Documents\Dreamtonics\Synthesizer V Studio\scripts`，其中 *** 为你的用户名。

## 停止所有运行中的脚本

强制停止所有运行中的脚本，该功能可能会导致一些脚本功能失效或崩溃。

:::tip
脚本的具体内容（包括官方脚本）请参考 [脚本下载界面](../../download/script.md)
:::
