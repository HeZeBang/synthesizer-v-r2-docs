## 吸附（编曲/钢琴卷帘）

控制音符的自动对齐程度，其中编曲吸附对应序列吸附，钢琴卷帘吸附对应音符吸附。

在新版中`1/n个四分音符`已被翻译为`m分音符`，他们的对应关系如下表：

| 1/n个四分音符 | m分音符（三连音） |
| ---- | ---- |
| 2 | 8 |
| 4 | 16 |
| 6 | 8-三连音 |
| 8 | 32 |
| 12 | 16-三连音 |
| 16 | 64 |
| 24 | 32-三连音 |
| 32 | 128 |

其中关闭吸附后你可以自由调整音符或序列。
